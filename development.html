Project Development

On the Hippo Forge, projects are maintained by Hippo. Hippo will not give push rights to external developers.

Code Contributions

On GitHub, code contributions exist in the form of pull requests. External organisations can always drive those pull requests, or feature requests, or release requests by contacting our services department.
