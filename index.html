Redirected here?
You may have been redirected here from forge.onehippo.org.
That Hippo SVN Forge has been discontinued on 16 March 2017 and followed up on GitHub at github.com/onehippo-forge
Artifacts previously in SVN can now be retrieved from the Hippo Maven Forge Repository at http://maven.onehippo.com/maven2-forge.

Welcome!

Welcome to the Hippo Forge on GitHub, the community place for plugins, add-ons and extensions built on top of Hippo CMS or Hippo DX.

Community plugins are maintained by the Hippo community and by implementation projects.
For instance, it is common for a plugin upgrade to be triggered by a project upgrade.

The projects

See the project list page for an overview of all projects at hosted at github.com/onehippo-forge.

Initially, 33 selected projects were migrated in March 2017 from the discontinued Hippo SVN Forge at forge.onehippo.org.

URL Rewriter exception
The URL Rewriter plugin has been so successful that it has migrated to Hippo GitLab instead of GitHub.
